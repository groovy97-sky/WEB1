CSS 배우기

1. style 태그와 사용자

meta 밑~ /head 사이에 style /style 태그 사용 margin-border-padding


2. 문자 정렬

정렬을 하기 위해서는 text-align:---; 을 사용한다. 문자이기 때문에 text가 들어간다. 이는 html이 아닌 CSS이다.

  • 가운데로 정렬은 { text-align: center; }

  • 왼쪽/오른쪽 으로 정렬은 { text-align: left/right; }

  • 양쪽으로 정렬은 { text-align: justify; }


3. 글자 색상

글자색상을 변경하기위해선 font color="색상" /font 이라는 간단한 태그를 사용해왔지만 html5 부터는 font 태그를 지원하지 않기 때문에 css인 style을 사용해 주어야 한다. font를 사용해도 적용은 되지만 웹표준은 아니게 된다. 그렇담 css style은 어떻게 사용해야 할까? 우선 내가 색을 바꿀 문자가 한 블럭단위의 문장인지 여러 문장의 일부분인지 확인해야 한다. 블럭이라면 div를 사용하고, 일부분이라면 span을 사용한다. div는 해당 문장이 자동으로 한줄 엔터가 된다. 문장을 붙여 이어가고 싶다면 span을 사용하면 된다. 만약 문장에서 사용하는 것이 아닌 head의 style에서 사용하고 싶다면 사용자를 만든 뒤 { color:색상; } 로 설정할 수 있다. 이때 링크를 건 문자의 색상이 자동으로 파란색으로 변경되는 것을 바꾸고 싶다면 사용자 a를 사용해야 한다.

  • a { color:색상; }

  • div style="color:색상;" /div

  • span style="color:색상;" /span


4. 링크 밑줄 삭제

만약 a href로 링크를 걸어주었다면 원치않는 밑줄이 생길 것이다. 이럴때에는 style에 사용자 a를 이용하여 하나 만들어 준다. 그 뒤 { text-decoration: none; } 을 입력해준다.

5. 테두리 그리기

테두리를 그리려면 border 부분을 건들여 줘야한다. 무조건적으로 border은 스타일을 지정해 주지 않으면 기본값인 none이 사용되어 테두리가 보이지 않기 때문에 스타일을 가장 먼저 지정해 주어야 한다. 스타일을 지정해주는 코드는 { border: solid; } 이다.
다음으로 테두리 너비를 정해주어야 하는데 코드는 { border: 20px;} 이다.
만약 테두리 색을 지정하고 싶다면 예상하는대로 { border: 색상; } 이다.
이제 테두리 위치를 정해주어야 한다. 그냥 border은 직사각형 테두리가 생겨나고 아래, 위, 왼쪽, 오른쪽은 border-bottom, border-top, border-left, border-right 순이다. 이렇게 해버리면 border: 를 중복해서 계속 입력해주어야 한다. 그래서 우리는 이렇게 간단하게 표기할 수 있다.

  • { border: 10px solid black; }

  • { border-bottom: 5px solid balck; }

  • { border-right: 2px solid red; }


6. 여백

테두리를 설정했는데 글의 여백이 한쪽으로 치우쳐 있다면 치우쳐지지 않게 수정하고 싶을 것이다. 이럴땐 margin와 padding부분을 건들여 주면 된다. 우클릭으로 검사 를 클릭한 뒤 어느부분이 문제인지 확인해 본다. 종류는 bottom, top, right, left 네종류로 같다. (-아무것도 없으면 직사각형 전체 영역) 너비와 높이의 폭을 조정하고 싶다면 width 와 height를 사용한다.

  • { margin:크기; }

  • { padding-bottom: 크기; }

  • { width:크기; }

  • { height:크기; }


7. 글씨 크기

앞서서 글 색상을 설정할때 font를 사용하지 않는다고 설정했는데 글씨 크기를 조정할 때에는 font가 사용된다.

  • { font-size:사이즈; }


8. class 사용하기

본인이 원하는 부분들만 같은 기능을 넣고 싶을 때 사용자 이름이 달라 곤란할 때가 있다. 그럴때는 class="클래스 이름" 를 태그안에 넣고 사용자를 .클래스 이름 을 사용하면 된다.

  • .이름{ font-size:사이즈; }

    class="이름">


9. id 사용하기

id는 고유의 이름이기 때문에 하나이상의 태그에 같은 id이름을 줄 수 없다. id이름이 하나 정해지면 다른 태그에 또 사용할 수 없는 것이다. id이름으로 사용자를 적을 때는 #아이디 이름{ } 을 사용한다.
이때 사용자에 id이름을 여러개를 적어 한번에 같은 효과를 줄 수도 있다.
만약 ol이 여러개가 있는데 그중 특정한 ol에만 효과를 주고 싶다면 #아이디 이름 ol 해서 해당 아이디에 가장 가까운 ol에 효과를 줄 수 있다.

  • #이름{ font-size:사이즈; }

    id="이름">

  • #이름1 #이름2{ font-size:사이즈; }

    id="이름1"> id="이름2">

  • #이름3 ol{ font-size:사이즈; }

    id="이름3"> ol>


10. div 사용하기

아무 태그가 없는 텍스트에도 효과를 주고 싶은데 태그가 없으니 곤란한 일이 있을 것이다. 이럴때 아무 효과가 없는 div /div 태그를 사용해 주면 된다. div는 여러부분에 많이 쓰이기 때문에 사용자로 사용하지 않는다. div를 사용하려면 class나 id를 줘서 사용자에서 사용하면 된다.

  • #이름{ font-size:사이즈; }

    div id="이름"> /div>


11. display 설정

display는 말 그대로 해당 사용자에 관하여 시각적으로의 표현을 설정해 준다. 종류는 4가지가 있으며 border의 soild none와는 기능이 살짝 다르다. border의 soild와 none은 테두리의 보여짐과 안보여짐을 나타낸다면 display의 block와 none은 사용자 자체의 보여짐과 안보여짐을 나타내는것이다. block은 보이는 것, none은 보이지 않음, inline은 인라인 박스, inline-block은 block와 inline의 중간 형태이다. 이때 block와 inline이 뭔가 할 수 있다. html의 태그들은 block와 inline으로 나눌 수 있다. block은 div와 비슷하고 inline은 span와 비슷하다고 생각하면된다. display를 굳이 설정해주지 않아도 기본적으로 block이 된다. display는 block보다는 none 기능을 나타낼 때 많이 사용한다.

  • { display: block; }

  • { display: none; }


12.grid 정렬
grid를 이용해 레이아웃을 설정하려면 레이아웃을 하고자 하는 블럭이 두개가 있을 경우 그 두개의 블럭을 감싸고 있는 div 를 만들어 주어야 한다. 이때 div로 사용자 이름을 사용하지 않기 때문에 id를 넣어주어야 한다. 만약 div id="grid" /div 로 감싸주었다면 사용자에서 #grid{ } 를 입력해주면 된다. 여기서 어떤 내용이 들어가야 하는지 알려주겠다.
우선 display는 위에서 4가지가 있다고 했는데 사실은 5가지로 grid가 있다. { display:grid; } 설정을 반듯이 해주어야 한다. 다음으로는 그 두가지 블록을 행으로 정렬할 지 열로 정렬할지 정해야 한다. 행으로 정렬하고 싶다면 { grid-template-rows: 값1 값2; }, 열로 정렬하고 싶다면 { grid-template-columns: 값3 값4; } 를 적어주면 된다. 이때 값도 중요하다. 사용자가 창을 늘리고 줄이는 대로 같이 폭이나 너비가 움직이게 하고 싶다면 fr의 값을 주면 된다. 블럭이 두개일 때 1fr 2fr 의 값을 주게 되면 전체적인 창을 3등분으로 나누어 1:2 비율을 가지게 되는것이다. 움직이지 않게 하고 싶다면 px로 값을 주면 된다. 참고로 px 와 fr은 섞어서 사용 가능하다. 값의 하나는 fr, 다른하나는 px로 주어도 된다는 말이다. 그렇게 되면 px는 움직이지 않지만 fr로 설정한 블록만 움직이게 될 것이다.
px와 fr의 큰 차이점은 px는 창을 줄이게 되었을때 글씨가 다음줄로 넘어가거나 하지 않고 아에 창에서 사라진다. 그리고 아래쪽에 좌우로 움직일 수 있는 바가 생기게 된다. fr은 아무리 줄여도 글씨가 사라지지 않고 다음줄로 넘어가게 되며 아래쪽에 좌우로 움직이는 바가 생기지 않는다.

  • { display:grid; } //기본 설정

  • { grid-template-rows: 값1px 값2fr; } //행 정렬

  • { grid-template-columns: 값3fr 값4fr; } //열 정렬


13. 반응형 웹 "미디어 쿼리" - 창의 크기

미디어 쿼리는 이러한 창의 크기가 어떠한 조건을 가질 때 이러한 반응을 보이도록 효과줄 수 있는 사용자이다. 괄호 안에 조건을 적고, 대괄호 안에 실행하는것을 적으면 되는데 이때 media사용자는 사용자 안에 사용자를 적을 수 있다. 즉, 이전의 사용자의 대괄호 안의 내용을 조건이 만족하였을 때 변경할 수 있다는 얘기다. 조건에서는 min와 max를 사용한다. min:800px일 경우 창의 크기가 최소800px 일때 이므로 창의 크기가 800px부터 대괄호를 실행한다는 내용이다. max:800px의 경우 창의 크기가 최대800px일 때이므로 창의 크기가 800px일 때까지 대괄호를 실행한다는 내용이다. 즉, 조건 까지 적용이 max, 조건 이후 부터 적용이 min 이다.

  • @media(조건){ }

  • @media(max-width:800px){ #grid{ display: block; } }

  • @media(min-width:800px){ h1{ border-bottom:none; } }


14. 중복 stylesheet 삭제 "link"
style부분이 여러 웹 파일들에게 동일하게 적용될 때 하나하나 복사 붙여넣기 해주어야 하는 불편함이 있다. 여기에 수정할게 하나 있다면 여러개의 파일을 하나하나 수정해 주어야 하는 것이다. 이러한 반복작업을 줄이기 위해 사용하는 것이 link 이다. style태그를 잘라 새로운 웹 파일을 하나 만들어 저장해준다.(이때 이름은 .css) 그 뒤 원래 파일로 돌아가 style자리에 link rel="stylesheet" href="이름.css" 을 입력해 준다. 그럼 이름.css 파일 하나만 바꾸면 link가 있는 모든파일에 적용이 되는 것이다.

  • 이름1.html file
    link rel="stylesheet" href="이름.css"

  • 이름2.css file
    style> /style
